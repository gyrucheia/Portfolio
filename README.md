# Personal Portfolio Website - Jamaica Ramores

### Project Overview
[cite_start]A professional portfolio developed for BSIT 2nd Year, demonstrating frontend integration, external API handling, and transaction-based features[cite: 49].

### APIs Used
1. [cite_start]**GitHub API**: Fetches live repository data from the `gyrucheia` profile to showcase technical projects[cite: 65].
2. [cite_start]**Leaflet.js & OpenStreetMap**: Displays a professional location map centered at Daet (14.114905, 122.947108)[cite: 65].
3. [cite_start]**EmailJS API**: Processes real-time contact form submissions to a personal Gmail account[cite: 65].

### Transaction Features
1. [cite_start]**Contact Form**: Includes JavaScript validation for name and email format, data processing via EmailJS, and visual success/error alerts[cite: 71].
2. [cite_start]**Appointment/Booking Request**: A simulation for scheduling creative sessions with date validation and user feedback responses[cite: 70, 71].

### How to Run
1. Open the project folder in **VS Code**.
2. Click **'Go Live'** using the Live Server extension to view the site in your browser.
